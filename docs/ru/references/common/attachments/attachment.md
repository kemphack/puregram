# `Attachment`

Класс **вложений**, от которого наследуются **все** остальные вложения.

```ts
import { Attachment } from 'puregram';
```

## Геттеры класса

### Содержание

* [`attachmentType`](#attachmenttype)

---

### `attachmentType`

**Возвращает тип вложения**

```ts
attachment.attachmentType // => AttachmentType | undefined
```
